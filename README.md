# Excel-Google-Sheets-analysis
Анализ данных с использованием Excel и Google Sheets
Описание проекта

Проект посвящен анализу данных о продажах и товарообороте с использованием Excel и Google Sheets.

В ходе работы были объединены данные из нескольких источников, проведен анализ продаж по территориям и неделям, рассчитаны ключевые показатели и подготовлены аналитические выводы.

Microsoft Excel
Google Sheets

INDEX / MATCH

Построение графиков
Выполненные задачи
Объединение данных
Объединены данные из нескольких листов в единую таблицу.
Выполнена проверка корректности объединения.
Анализ территорий
Рассчитан общий товарооборот по территориям.
Определены территории-лидеры по доле продаж.
Выявлены территории с наибольшим складским товарооборотом.
Анализ динамики
Построена недельная динамика продаж.
Проведен анализ изменения товарооборота во времени.
Выполнено сравнение показателей по периодам.
Результаты
Выявлены территории с максимальным вкладом в общий товарооборот.
Определены основные тенденции изменения продаж по неделям.
Подготовлены аналитические таблицы и визуализации для принятия решений.
