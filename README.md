# Excel Business Analysis

## Описание проекта

Проект выполнен с использованием Microsoft Excel и Google Sheets и включает несколько бизнес-кейсов по обработке, анализу и визуализации данных.

Основная цель проекта — продемонстрировать навыки подготовки данных, построения аналитических отчетов, работы со сводными таблицами и поиска бизнес-инсайтов.

## Используемые инструменты

* Microsoft Excel
* Google Sheets
* Pivot Tables
* VLOOKUP / XLOOKUP
* INDEX / MATCH
* Формулы агрегации
* Условное форматирование
* Диаграммы и визуализация данных

## Выполненные задачи

### Анализ продаж по территориям

* Объединены данные из нескольких листов в единую таблицу.
* Построена сводная таблица по территориям и неделям.
* Выполнен анализ товарооборота.
* Определены ТОП-3 территории по доле в общем товарообороте.
* Определены ТОП-3 территории по складскому товарообороту за последнюю неделю.
* Построена динамика продаж по неделям.

### Структурирование и очистка данных

* Выполнено преобразование неструктурированных данных в табличный формат.
* Добавлены справочники и классификаторы.
* Выполнена сегментация клиентов по объему трафика.
* Построены сводные таблицы с детализацией по категориям клиентов.

### Анализ и визуализация

* Созданы интерактивные фильтры.
* Построены аналитические диаграммы.
* Выполнено сравнение показателей между сегментами данных.
* Подготовлена итоговая отчетность для принятия решений.

### A/B Testing

* Выполнено распределение клиентов по тестовым группам.
* Проверена корректность распределения по регионам и городам.
* Проведен анализ результатов тестирования.
